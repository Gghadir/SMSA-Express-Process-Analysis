# SMSA-Express-Process-Analysis
This repository highlights a collaborative process analysis project focused on improving the logistics and supply chain operations of SMSA Express. As the project leader, I directed a team effort to dissect the existing delivery workflows, uncover hidden bottlenecks, and design a more efficient, practical way of operating.

## What We Accomplished
- __Current State Analysis:__ Led the team in mapping the existing logistics pipeline. We developed a comprehensive As-Is model using BPMN 2.0 to establish a clear baseline of current operations.
- __Problem Identification:__ Facilitated rigorous root-cause analysis sessions to pinpoint exactly where and why delays were happening, identifying key structural inefficiencies and redundancies.
- __Future State Design:__ Directed the engineering of an optimized To-Be model. We restructured the workflow to eliminate non-value-adding steps, aiming to significantly reduce cycle times and improve operational flow.

## Key Outcomes
- __Faster Cycle Times:__ The redesigned process compresses the time required for internal processing, directly accelerating overall delivery speed.
- __Better Efficiency:__ By cutting out redundant tasks, we created a framework where operational resources can be utilized much more effectively.
- __Actionable Strategy:__ Delivered a practical, data-backed plan that management can use to implement these structural changes seamlessly.
